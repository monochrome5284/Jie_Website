箭頭函式 (arrow function)

箭頭函式 (arrow function)

箭頭函式是 ES6（ECMAScript 2015）中引入的一種新的函式定義語法，它具有簡潔明瞭的語法和特殊的行為。以下是箭頭函式的基礎說明：

語法結構：

  • const 或 let：用於宣告函式的變數，通常使用 const 以確保函式的不可變性。
  • functionName：函式的名稱，可以是任何合法的變數名稱，通常根據函式的功能來命名。
  • parameters：函式的參數列表，如果沒有參數，則使用空括號 ()。
  • =>：稱為箭頭運算子，用於分隔參數列表和函式主體。
  • 函式主體：包含在大括號 {} 內的程式碼塊，表示函式的實際操作。

特點：

  • 省略 return 關鍵字：當函式主體僅包含一條語句時，可以省略 return 關鍵字，且該語句的值將自動成為函式的返回值。
  • 保留外部的 this：箭頭函式不會創建自己的 this 上下文，而是繼承外部的 this。這意味著在箭頭函式內部，this 將指向函式被創建時的上下文。
  • 無法用作構造函式：箭頭函式無法使用 new 關鍵字來創建新的物件實例，因此不能用作構造函式。
  • 無法變更 this：由於箭頭函式不具有自己的 this 上下文，因此無法使用 call()、apply()、bind() 等方法來變更 this。

when2meet使用方式

▎使用他人建立好的When2meet

▶ 第一步，輸入自己的名稱

When2meet優點在於不用註冊即可使用。
雖然有登入畫面，但僅需輸入自己的名稱Your Name即可登入Sign In。
名稱是方便調查所有人的時間，了解該時段誰可參加，誰無法參加。

▶ 第二步，選擇可以的時間

登入Sign In後，會出現下方畫面

左側粉色底表格

表示自己可以參與的時間（綠色儲存格）以及不能參與的時間（粉色儲存格）。

如何更改儲存格顏色
像Excel一樣選擇並拖曳框選多個儲存格，被選擇的儲存格們將改變顏色（原本粉色→綠色; 原本綠色→粉色）。

右側白色底表格

表示目前填寫人的時間統計結果

▶ 第三步，查看所有人可以的時間

  • 圖例
    依顏色表示下方時間表內特定時間段可參與的人數。
    系統用階層綠色表示，並且月深色的綠表示該時間段越多人可以參加。
  • 時間表
    顯示統計所有成員的時間調查結果。
    若滑鼠移至時間表上，將會於左側顯示所選時間以及該時間的所有成員可否參與之狀況。

▎自己創建When2meet

▶ 設定活動名稱及調查時間


如上圖所示功能：

  • 活動名稱
    可隨意命名，會在建立好when2meet後作為網頁title

  • 統計時間類型：

    • 特定日期（調查日期）

    • 不特定日期（調查星期）

  • 調查每日時間
    設定調查時間：每日上午xx:xx到晚間xx:xx。
    避免最後會議/活動時約在很奇怪的時間。
    ex:05/09 凌晨1點～3點

  • 時區
    預設應該是在台北
