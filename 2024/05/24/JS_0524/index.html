箭頭函式 (arrow function)

箭頭函式是 ES6（ECMAScript 2015）中引入的一種新的函式定義語法，它具有簡潔明瞭的語法和特殊的行為。以下是箭頭函式的基礎說明：

語法結構：

  • const 或 let：用於宣告函式的變數，通常使用 const 以確保函式的不可變性。
  • functionName：函式的名稱，可以是任何合法的變數名稱，通常根據函式的功能來命名。
  • parameters：函式的參數列表，如果沒有參數，則使用空括號 ()。
  • =>：稱為箭頭運算子，用於分隔參數列表和函式主體。
  • 函式主體：包含在大括號 {} 內的程式碼塊，表示函式的實際操作。

特點：

  • 省略 return 關鍵字：當函式主體僅包含一條語句時，可以省略 return 關鍵字，且該語句的值將自動成為函式的返回值。
  • 保留外部的 this：箭頭函式不會創建自己的 this 上下文，而是繼承外部的 this。這意味著在箭頭函式內部，this 將指向函式被創建時的上下文。
  • 無法用作構造函式：箭頭函式無法使用 new 關鍵字來創建新的物件實例，因此不能用作構造函式。
  • 無法變更 this：由於箭頭函式不具有自己的 this 上下文，因此無法使用 call()、apply()、bind() 等方法來變更 this。
